Where are the network disks and network files stored?

Different source files are stored in the corresponding directory

Add subtitles to video?

On the video playback page, click the subtitle icon, jump to the subtitle folder, and select the subtitle to be added.
Note: Currently only subtitle files are supported from the subtitles folder

After modifying the picture, the thumbnail of the file list still shows the old picture?

After clicking clear cache in settings, refresh the page again

Transfer files from computer

  1. Click the More button in the upper right corner
  2. Click Import from
  3. Select the computer

netdisc

  1. Authorize your online disk account
  2. Find the file you want to download
  3. Click the more action button at the bottom, or long press the file
  4. Select Save

Player gestures

Operation Gesture
Hide/Show control interface Click on the player
Fast forward and rewind Swipe left and right
Change brightness Swipe left up and down
Change volume Swipe right up and down
Screenshots/Video clips/GIFs Tap the scissors icon
Double speed Click the double speed icon on the right
Fast forward 2 times Long press the player
The screen is covered Pinch with two fingers

Other questions or suggestions

Click Support in Settings and select Send Feedback
