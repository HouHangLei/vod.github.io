    Download files in AVOD network

    Video file:
    1. Wait for the page to load, and the play button is displayed in the lower right corner (or click the player background in the webpage)
    2. Click the play button in the lower right corner
    3. Select the action after the tab pops up

    No play button in the bottom right corner of a webpage that contains a video?
    Please try again using

    1. Click the video in the webpage to play
    2. Or refresh the current page (click the more button on the right at the bottom, click refresh and try again)
    3. Save the current web page to bookmarks and reopen from bookmarks
    4. Still no play button? Please send an email to avod.suggest@gmail.com (long press to select New email message) and send us this URL.
    Other files:

    When clicking on other file links, it will automatically jump to the download page. you can copy
    https://www.browserdocumentsapp.com/demo/
    Go to the web page for access experience.

    Download web files

    Paste the link of the network file into the input box of the network, and click the forward button on the keyboard to automatically jump to the download page.
    Supported formats:
    Video (mp4/avi/m3u8/m4v/mov/f4v…)
    Audio (mp3/mpeg/mpeg-4/midi/wma…)
    eBook (txt/azw3/mobi…)
    Other (zip/pdf/jpg/png/doc/xls/ppt/docx…)

    Download M3U8 file

    When the M3U8 file is about to be downloaded and cannot continue to download. Please click Pause and then click again to start the download. If there is still a problem, please swipe right -> copy the link, then re-add the download task, and try to keep AVOD in the foreground.

    Transfer files from computer

    1. On the Connections tab, click Computer
    2. Copy the link and open it in the computer browser
    3. Upload the file in the computer browser
    4. Note: Please ensure that the mobile phone and the computer are connected to the same WIFI

    Baidu Netdisk

    1. Log in to your Baidu account
    2. Find the file you want to download
    3. Click the download button on the right to start the download

    Player Gesture Operation

    Operation Gesture
    Hide/Show Controls Tap Player
    Fast forward and rewind Swipe left and right
    Change brightness Swipe left up and down
    Change volume Swipe right up and down
    Screenshots/Video clips/GIFs Tap the scissors icon on the right
    Double speed Tap the more action icon in the upper right corner
    Fast forward 2 times Long press the player

    How to return to the desktop from a web page

    Click the More button at the bottom right and click “Return to Desktop”
